# Notebooks

This directory contains the main Jupyter Notebook used for the loan approval analysis and modeling.

---

## File

- **loan_prediction_code.ipynb**

---

## Contents

The notebook includes the complete data science workflow:

### 1. Data Cleaning

- Handling missing values
- Fixing inconsistencies
- Data type conversions
- Log transformations for skewed variables

---

### 2. Exploratory Data Analysis (EDA)

- Distribution analysis
- Categorical variable analysis
- Approval rate comparisons
- Statistical summaries

---

### 3. Statistical Testing

- Chi-square tests to evaluate relationships between:
  - Credit history and loan approval
  - Education and loan approval
  - Property area and loan approval

---

### 4. Feature Engineering

- Creation of Loan Term Category
- Encoding of categorical variables
- Preparation for modeling

---

### 5. Model Building

- Logistic Regression
- Random Forest
- XGBoost

---

### 6. Model Evaluation

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
- Confusion matrix

---

### 7. Model Interpretation

- Feature importance analysis
- Comparison of model performance

---

## Purpose

This notebook demonstrates an end-to-end machine learning pipeline for loan approval prediction, combining statistical analysis with predictive modeling.
