# Dashboard

This directory contains the dashboard developed for the loan approval analysis project.

---

## File

- **dashboard.pbix**

---

## Overview

The dashboard provides an interactive visualization of key insights derived from the dataset, supporting both exploratory analysis and business interpretation.

---

## Key Features

- Loan approval distribution analysis
- Approval rates by:
  - Education level
  - Marital status
  - Property area
- Credit history impact on loan approval
- Loan term category analysis
- Integrated insights using visual tools (e.g., decomposition tree)

---

## Purpose

The dashboard translates analytical findings into business-friendly insights, enabling:

- Better understanding of loan approval patterns  
- Identification of key factors influencing decisions  
- Support for data-driven decision-making  

---

## Notes

- Data preprocessing was performed prior to visualization
- The dashboard complements the machine learning and statistical analysis
- Designed for interactive exploration of loan approval trends
