# Raw Data

This directory contains the original loan application dataset used in the project.

---

## File

- **loan_dataset.csv**

---

## Description

The dataset includes demographic, financial, and credit-related information about loan applicants, such as:

- Applicant income
- Co-applicant income
- Loan amount and term
- Credit history
- Education level
- Marital status
- Property area
- Loan approval status

---

## Notes

- This dataset has not been modified
- It may contain missing values and inconsistencies
- It serves as the starting point for all preprocessing and analysis steps

Maintaining the raw dataset ensures reproducibility and allows validation of all transformations applied later in the project.
